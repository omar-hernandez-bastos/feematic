## Getting Started

### Install

Install dependencies.

```bash
npm install
```

Serve with hot reload at http://localhost:3000.

```bash
npm run dev
```

### Lint

```bash
npm run lint
```

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```
